Instructions on how to use Blastoserve Cam

Intro about software :

Blastoserve cam is open source camera desktop app for windows ,linux & mac. It is absolutely free of cost ,easy to use & with cool UI . This app simply do the patient registration like you have to simply install it and then go and open the blastoserve cam app then you got simple UI. Onclick of start collection of data you start the application and start the first page by filling the form . In the form you have to type some patient information and then click on the start capturing data for patient and then you got new page in that you have to first select the camera and then capture some images if you want too, and write some comments in below images and you can start the recording also you can puase the recording . This all data is saved automatically in local system .

What to do next :

Now you install blastoserve cam on your system . Now your first question might be what to do next ? . You have to simply open the the application by double clicking blastoserve cam icon from your desktop . Now you got simple UI which contain one advertisment video and after advertisment video you got one button which have name is "start collection of data". By pressing of this button you can start the application which means you redirected to the next page which can registered a patient with some information . Now you on patient form, fill this form with patient details . and then click on green button which contain the name "start capturing data for <patient name>". By clicking on this green button you got new page which is our core of this is application . In this page you got :

  • On the top center you got patient name and in bracket you got the type of procedure.

  • In below that you got "select camera : " button . select your camera from this dropdown .

  • In below that you got 4 button at left side and camera-view at right side .

    • First button is green button which containing the name "Click here or Click on image to take Photos" . The work of this button is to capture image and display image at end of the screen .

    • Second button is red button which containing the name " Start Recording" .which have work to record the video of patient . after the recording started this button got name stop recording and onclick of stop recording we stop recording as well as we can display it at the end of the screen. It have some optional feature like pause recording and play recording .You can press this button according to your choice.

    • Third button is white button which containing the name "Saved Data and Collect Data For New Patient" . As accoding to name this button can collect the data of this page and store it on your system as a files . And this button also redirect to the patient-form page.

    • Fourth button is also white button which containing the name "Save Current Data and Moved to first page" . When you click on this button data is saved automatically to your local setup folder where you stored the setup file. and moved to the advertisment page that is our first page.

  • And lastly all the images and videos are displayed according to the time when they are captured

Where the data get saved ? :

In your local system, Where you store setup file during installation of this software, Open the setup file in that you got patient-details folder. In the folder you will find the registered patient name,and currenct date and time as folder name. In that folder you will find the patients name ,images and comments. The first file is patient_name .txt file in that txt file the patient information is displayed . and then some images and videos are displayed on the folder . and in next to the image you will find the same name with .txt file which contains the image comments and name of the image . and same for all images and video also.
